커리어 도구

취업/이직 준비를 위한 무료 도구. 서버 없이 브라우저에서 바로 실행돼요.

이 페이지는 브라우저 로컬 저장소만 사용해요. 입력한 JD/이력서 텍스트는 공개 서버로 전송되지 않고, 같은 브라우저에서만 이어서 볼 수 있어요.
분석 결과를 더 입체적으로 보려면 데모 War Room에서 기업 상세 정보를 함께 확인하세요.
🔍

JD 분석기

채용공고 URL 또는 텍스트를 붙여넣으면 자동으로 기술 요구사항, 경력 수준, 핵심 역량을 분석해요

채용공고를 입력하고 분석 버튼을 눌러주세요

📋

이력서 점수 체크

이력서만 있으면 바로 분석해요 — 회사 선택 없이도 완성도 점수, 개선 포인트, ATS 최적화 팁, 경력 회사 심층 분석까지 한 번에 제공해요

이력서 파일을 드래그하거나 클릭하세요

.txt, .md, .pdf 지원

또는

이력서를 붙여넣고 점수 체크를 눌러주세요

✍️

자소서 클리셰 감지

자소서에서 식상한 표현을 찾아내고 대안을 제안해요

자소서를 붙여넣고 감지 버튼을 눌러주세요

⚖️

기업 비교기

관심 기업 2–3개를 선택하면 5개 차원으로 비교하고 추천 순위를 알려줘요

기업을 2개 이상 선택하면 비교 결과가 나타나요

더 강력한 분석이 필요하다면

로컬에서 전체 기능을 사용해보세요
