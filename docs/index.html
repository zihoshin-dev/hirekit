HIREKIT

기업을 분석하고, 공고를 매칭하고, 면접을 준비하는 가장 선명한 터미널 월드예요.

4~8시간짜리 기업 리서치를 명령어 하나로 끝내는 HireKit을, 콘솔에서 모험 퀘스트를 깨듯 읽히는 픽셀 2D 랜딩으로 다시 구성했어요. 8개 데이터 소스, 가중 스코어카드, 맞춤 면접 준비 흐름을 한 장에 담았어요.

Core Quests

합격에 필요한 흐름을, 픽셀 RPG의 퀘스트 슬롯처럼 또렷하게 묶었어요.

첫 리서치부터 최종 면접 준비까지 이어지는 HireKit 핵심 명령어를 게임 UI 카드처럼 재배치해서 한 번에 훑을 수 있게 했어요.

Core Quest

기업 분석

🔍

DART 재무 데이터, 뉴스, GitHub 기술 성숙도, 조직 문화 신호를 모아 12개 섹션 리포트를 생성해요.

hirekit analyze 카카오 --no-llm -o terminal
Target Match

공고 매칭

🎯

채용 공고를 붙여넣으면 적합도, 스킬 갭, 지원서에서 강조할 프로젝트를 빠르게 정리해줘요.

hirekit match --jd job.txt --company kakao
Boss Prep

면접 준비

🗣️

실제 기업 맥락과 최신 뉴스를 반영해 맞춤 질문을 만들고, 답변 준비 포인트까지 이어줘요.

hirekit interview 카카오
Loadout

이력서 리뷰

📄

직무 대비 점수를 계산하고 키워드 정합성, 성과 문장, ATS 대응까지 실질적인 개선 포인트를 잡아줘요.

hirekit resume --file resume.md --company kakao
Story Forge

자기소개서

✉️

기업의 비즈니스 과제와 가치관을 반영해서 범용 템플릿이 아닌 맥락 중심 초안을 만들어요.

hirekit coverletter --company kakao --resume resume.md
Plugin Bay

소스 확장

🔌

커스텀 데이터 소스를 등록해 원하는 신호를 더 붙일 수 있어서, 분석 맵을 계속 넓혀갈 수 있어요.

@SourceRegistry.register("my_source")
Data Nodes

8개 데이터 소스를 지도처럼 깔아두고, 어디서 어떤 신호를 가져오는지 한눈에 보이게 했어요.

공식 공시, 뉴스, 검색, 후기, AI 시맨틱 검색을 한 장의 픽셀 네트워크 보드처럼 묶어서 HireKit이 왜 빠르고 믿을 만한지 바로 읽히게 했어요.

DART

DART_API_KEY

Custom / Plugin

Your Source

Python 몇 줄로 자신만의 데이터 소스를 꽂아 개인화된 분석 루트를 만들 수 있어요.

@SourceRegistry.register
Live Readout

데모 구간은 “결과가 어떻게 읽히는지”에 집중해서, 실제 CLI의 속도감과 스코어 보드를 함께 보여줘요.

픽셀 스타일을 입혔지만 핵심은 여전히 데이터예요. 그래서 콘솔 로그, 결과 개수, 점수 요약이 가장 먼저 보이도록 배치했어요.

zsh // mission: analyze

Why It Works
  • 8개 데이터 소스를 한 번에 묶어서 리서치 시간을 줄여요.
  • 100점 스코어카드로 직관적인 비교가 가능해요.
  • 면접 준비 흐름까지 이어져서 결과가 다음 액션으로 연결돼요.
Quickstart Route

처음 쓰는 사람도 “설치 → 설정 → 첫 분석” 흐름을 따라가면 바로 시작할 수 있게 만들었어요.

카드마다 명령어를 바로 넣어둬서, 읽는 동시에 복사하고 실행할 수 있는 튜토리얼처럼 동작해요.

01

설치

Python 3.11 이상이면 바로 설치할 수 있어요. 필요하면 LLM 확장도 이어서 붙일 수 있어요.

pip install hirekit
02

설정

대화형 설정으로 DART, GitHub, Naver News 같은 소스를 빠르게 켜고 꺼요.

hirekit configure
03

첫 분석

국내외 기업을 바로 분석하고, 결과를 터미널이나 Markdown 리포트로 확인해요.

hirekit analyze 카카오
Progress Map

로드맵도 픽셀 월드맵처럼 보이게 해서, 지금까지 완성한 것과 앞으로 열릴 지역이 구분되도록 했어요.

완료된 기능은 안정적인 그린, 진행 중인 항목은 골드, 다음 확장은 블루로 나눠서 제품의 방향성이 시각적으로도 이어지게 구성했어요.

Phase 1 / Complete

기반 구축

  • DART 전자공시 연동과 GitHub 기술 성숙도 점수화를 붙였어요.
  • 멀티소스 뉴스 수집과 5개 차원 가중 스코어카드를 만들었어요.
  • 구조화 Markdown 리포트로 결과를 저장하게 했어요.
Phase 2 / Live

취업 지원 도구 세트

  • 스킬 갭 분석 포함 공고 매칭 흐름을 추가했어요.
  • 면접 준비, 이력서 리뷰, 자기소개서 작성까지 이어졌어요.
  • 랜딩에서는 이 흐름을 퀘스트 카드처럼 이해하기 쉽게 정리했어요.
Phase 3 / Next

확장과 커뮤니티

  • SEC Edgar 연동으로 미국 기업 분석 지원을 넓혀요.
  • CLI를 쓰지 않는 사용자를 위한 Web UI 흐름을 준비해요.
  • 커뮤니티 플러그인 마켓플레이스와 PyPI 정식 릴리스를 목표로 해요.

오픈소스 커뮤니티와 함께 맵을 확장해요.

새로운 데이터 소스, 리포트 템플릿, 다국어 지원, Web UI까지 어떤 기여도 환영해요. 이 랜딩도 같은 정신으로, 제품의 정보 구조를 더 생생하게 보여주기 위해 다시 그렸어요.
